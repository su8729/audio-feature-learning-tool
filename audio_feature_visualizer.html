Frequency spectrum / FFT

FFT decomposes a signal into sinusoidal components. It reveals which frequencies exist, but not when they occurred.

Focus: global frequency content
What to notice
Peaks and sidebands

AM and FM signals are great for seeing spectral structure beyond a single tone.

Core formula
Δf = fs / N

Larger N narrows each bin, improving frequency resolution.

New toggle
Linear vs log magnitude

Log scale reveals weaker harmonics and quieter components.

MFCC, Δ, and ΔΔ

This section computes real Mel filter bank energies, real MFCC via DCT, and temporal derivatives. It connects compact static features with change over time.

Focus: compact ML-friendly features
What to notice
MFCC captures spectral envelope

It compresses broad shape instead of preserving every frequency bin.

Delta idea
Δ tracks first-order change

If the sound is evolving, delta coefficients highlight that motion.

Delta-delta idea
ΔΔ tracks acceleration

This is useful when the rate of change itself changes.
